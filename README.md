A simple, no-fuss CLI tool to convert videos to GIF or WebM.

## Why?
Sometimes you just need a GIF, and you don't want to open a heavy video editor.

## Requirements
- ffmpeg

## Usage
```bash
./v2a.sh my-video.mp4 gif
./v2a.sh my-video.mp4 webm
```
